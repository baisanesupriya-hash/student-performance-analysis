# Student Performance Analysis & Prediction

Data analysis and machine learning project to predict student performance using Python.Data analysis and machine learning project to predict student performance using Python. 


## Project Overview

This project analyzes student performance data to understand how different factors such as study hours, attendance, and previous scores influence final exam results.

A machine learning model is also built to predict student scores based on these features.

---

## Objectives

* Perform exploratory data analysis (EDA)
* Visualize relationships between variables
* Build a machine learning model
* Evaluate prediction performance

---

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

---

## Dataset

The dataset contains information about student academic behavior and performance.

Example features:

* Study Hours
* Attendance
* Previous Scores
* Participation in Activities
* Final Score (Target Variable)

---

## Project Workflow

### 1. Data Loading

The dataset is loaded using Pandas for further analysis.

### 2. Data Exploration

Basic information about the dataset such as structure, summary statistics, and missing values is analyzed.

### 3. Exploratory Data Analysis

Visualizations such as scatter plots and correlation heatmaps are used to understand relationships between variables.

### 4. Model Building

A machine learning model is trained using the dataset.

### 5. Model Evaluation

The model is evaluated using the R² score to measure prediction accuracy.

Model Accuracy: **~73%**

---

## How the Model Predicts Student Scores

The machine learning model is trained using historical student data that includes study hours, attendance, and previous academic scores.

During training, the model learns patterns and relationships between these input features and the final exam score.

Once trained, the model can predict the final score of a student by analyzing their study habits and past performance. The prediction is generated using the learned relationships between the features and the target variable.

The model achieved an R² score of approximately 73%, indicating that it can explain a significant portion of the variation in student performance.

---

## Results

The model successfully identifies patterns in student performance and predicts final scores with reasonable accuracy.

## Conclusion

This project demonstrates how data analysis and machine learning techniques can be used to analyze educational data and predict academic outcomes.
